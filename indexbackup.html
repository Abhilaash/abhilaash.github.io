Hackathon Projects:

Recipe Project: A website where you can input ingrediants and then find a recipe for something you can make. It has a functional login system that sores all of your usernames and passwords using a secure system in a database.

Need2Meet: Having difficulties scheduling a meeting with many friends? This website finds a public meeting area that is most convinient for all of the members as well as finding the best time by using a heuristic based off the inputted times and locations

PlanIt: Plans what assignment to complete when using a heuristic based off various factors such as your grade in a class, how much you like a class, and the estimated length of the assignment.

Web Application Development Labs:

Canvas Lab: Click points on the canvas, toggle edges between nodes, select nodes, and delete them. Also, save the file and upload the text to resume.

Capitals Lab: Think you can name every state's capital in 5 minutes? You can skip a state if you don't know it, but it will come back, so be ready! Similar to the states lab, but a state is displayed, and you must type in that state's capital.

Identifying Item Elements Lab: This lab identifies what item is clicked on.

Permutations Lab: This lab takes the inputted word and finds and displays every possible permutation of that word's letters without repeating words. It counts the number of permutations too.

States Lab: Think you can name all 50 states in 5 minutes? The counter starts when you start typing. If a state is entered, it will automatically clear it off.

Summer Assignments Lab: What I did over my summer from sophomore to junior year

Time Lab: Gives the EST Time. Checks when the page was previously entered by anyone. Displays the number of times this page has been viewed.

Webscrape Lab: Finds the weather in the given address by scraping the weather off of wunderground. Accepts zipcodes or address. Displays all previously checked data. This used to be a cron job that could serve research purposes by pulling the data once every 24 hours. The cron job had to be stopped because of issues with repeated requests and the database taking up too much memory.
